# cra-template-solana-typescript

A TypeScript [Create React App](https://github.com/facebook/create-react-app) template for [Solana](https://solana.com/).

To use this template, add `--template cra-template-solana-typescript` when creating a new app.

For example:

```sh
npx create-react-app my-app --template cra-template-solana-typescript

# or

yarn create react-app my-app --template cra-template-solana-typescript
```

For more information, please refer to:

- [Getting Started](https://create-react-app.dev/docs/getting-started) – How to create a new app.
- [User Guide](https://create-react-app.dev) – How to develop apps bootstrapped with Create React App.
